# 🧮 Calculadora de IMC

Calculadora de Índice de Massa Corporal feita em Python com auxílio do Cursor.

## 🔄 Como funciona

1. Informe seu peso em kg
2. Informe sua altura em metros
3. O programa calcula seu IMC e exibe a classificação:
   - Abaixo de 18.5 → Abaixo do peso
   - 18.5 a 24.9 → Normal
   - 25 a 29.9 → Sobrepeso
   - 30 ou mais → Obeso

## 🛠️ Tecnologias

- Python
- Cursor (editor com IA)
